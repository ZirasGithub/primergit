Comandos basicos git

comandos basicos en órden:

  1. git init
    2. git status
  2. git add (git add . (cumple la función de guardar los cambios))
  3. git commit (git commit -m "Primer commit")
    4. git remote git push (git push -u origin main) git remote add (crea una conexion a un repositorio remoto git remote add origin *link del remoto*)
